# Secure_kit

SecureKit
A minimal password utility built in vanilla JavaScript. Two tools in one page: a real-time password strength checker and a password generator.
Features

Password Checker — checks against 5 requirements in real time as you type: length, uppercase, lowercase, number, and symbol
Password Generator — generates a random password between 8 and 64 characters using a slider

Tech Stack
JavaScript · HTML · CSS · DM Mono (Google Fonts)
